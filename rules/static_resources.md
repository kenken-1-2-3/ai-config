# Static Resources Rules

Apply these rules to `static-resources`.

## Scope

- Only add, replace, or remove assets named by the user or required by the task.
- Do not optimize, rename, recompress, or convert unrelated assets.
- Preserve existing folder naming and asset naming conventions.

## Verification

- Confirm changed asset paths and file types.
- Avoid broad asset scans unless needed to locate the requested resource.
